# Titanic-Survival-Analysis-Exploring-Factors-Affecting-Passenger-Survival
An exploratory data analysis project uncovering key factors that influenced survival in the Titanic dataset using Python and data visualization.

# 🚢 Titanic Survival Analysis

## 📌 Objective

The objective of this project is to perform exploratory data analysis (EDA) on the Titanic dataset to identify the key factors that influenced passenger survival.

---

## 📊 Dataset

The dataset contains information about Titanic passengers, including:

* Age
* Gender
* Passenger Class (Pclass)
* Fare
* Survival status

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 🔍 Data Cleaning Process

* Handled missing values using mean, mode, and forward fill methods
* Converted categorical data where necessary
* Removed inconsistencies and ensured data quality

---

## 📈 Data Visualization

Various visualizations were created to understand patterns and relationships:

* Age distribution histogram
* Survival count plots
* Gender vs survival comparison
* Passenger class analysis
* Correlation heatmap

---

## 💡 Key Insights

* Female passengers had a higher survival rate compared to males
* Passengers in 1st class were more likely to survive
* Higher fare passengers showed better survival chances
* Age had a slight impact on survival

---

## 📌 Conclusion

The analysis shows that survival on the Titanic was strongly influenced by gender, passenger class, and socio-economic status. These insights highlight how data analysis can uncover meaningful patterns in historical datasets.
